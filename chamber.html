Chamber

For this project, I decided to build an informational website about the dangers of algorithms in the digital world.

There is an interactive page where the user can click on a button describing an interest and see what the site would generate based on the interest. The goal is to show the user how algorithms and companies that use algorithms to push a product can cause harm and insecurites.

Html, css, and JavaScript was used to build the webstie along with libraries such as glide.js, aos, and animate.css to make the site feel more fluid and interactive.
