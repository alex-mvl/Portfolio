Interactive Storybook

This project is a digital narrative experience that combines storytelling with interactive elements. Users can engage with the story by interacting with characters, objects, or environments, which provide visual and audio cues for the user to follow throughout the story. Through animations, sound, and visual cues, the storybook creates a visually captivating and interactive storytelling environment that is web browser accessible.

This interactive storybook was built using the p5.js library for JavaScript. The storybook integrates multimedia elements such as images, text, audio, and animations. Interactivity is achieved through event handling and user input detection, allowing readers to interact with trigger scripted events.
